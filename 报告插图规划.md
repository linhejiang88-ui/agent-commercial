# 报告插图规划

**日期：** 2026 年 4 月 28 日

**用途：** 规划主市场分析报告中适合插入图片的位置、图片类型和后续制图任务。

**适用报告：**

- `2026年软件及AI-Agent外包及咨询市场分析报告（插图版）.md`

## 1. 插图原则

- 优先服务核心判断：突出 `AI 咨询 + AI Agent 实施 + 托管运营` 是最大机会。
- 每个一级章节最多放 1-2 张关键图，避免把报告做成图表堆砌。
- 数据图必须能追溯到正文已有来源；分析图可以基于正文判断整理，但要标注为“分析框架”。
- 插图优先服务主报告的核心结论，避免为已删除的重复草稿保留额外维护口径。

## 3. 按章节整理的插图需求

### 3.1 摘要

#### 图 1：核心机会排序图

- **建议标题：** 2026 年 3 人团队技术外包机会排序
- **推荐位置：** `## 摘要` 末尾、正文第一章之前。
- **图表类型：** 横向优先级条 / 金字塔 / 排名卡片。
- **图中内容：**
  - 最大机会：AI 咨询 + AI Agent 实施 + 托管运营
  - 次优机会：BPO 流程自动化 + 存量系统 AI 化改造
  - 可做机会：AI 辅助数据分析 / KPO 增强
  - 低优先级：纯软件开发、人天外包、简单 Bot
- **后续制图任务：**
  - 不需要新增数据，直接基于摘要结论生成分析图。
  - 主报告已使用该图。

### 3.2 市场规模与增长分析

#### 图 2：市场规模与 CAGR 对比图

- **建议标题：** 软件外包、AI Agent 与 AI 咨询市场规模及增速对比
- **推荐位置：** `1.1 市场规模数据` 之后。
- **图表类型：** 组合柱状图 + CAGR 标注，或气泡图。
- **核心数据：**
  - 软件开发外包：2026 年约 6183.8 亿美元，2031 年约 9770.4 亿美元，CAGR 约 9.60%。
  - AI Agent：2026 年约 118 亿美元，2035 年约 2224 亿美元，CAGR 约 38.6%。
  - AI Consulting Services：2026 年约 140.8 亿美元，2035 年约 1168 亿美元，CAGR 约 26.49%。
- **图中重点：**
  - 软件外包规模最大，但增速较低。
  - AI Agent 增速最高。
  - AI 咨询规模小于软件外包，但更贴近新增预算和落地服务。
- **后续制图任务：**
  - 统一美元单位。
  - 标注不同预测周期，避免误导为同一时间跨度。

#### 图 3：中国 ITO / BPO / KPO 走势对比图

- **建议标题：** 2026 年初中国离岸服务外包结构分化
- **推荐位置：** `1.1.2 中国市场` 后。
- **图表类型：** 分组柱状图。
- **核心数据：**
  - 2026 年 1-2 月离岸 ITO 执行额 476.2 亿元，同比下降 7.8%。
  - 2026 年 1-2 月离岸 BPO 执行额 238.4 亿元，同比增长 12.4%。
  - 2026 年 1-2 月离岸 KPO 执行额 340.4 亿元，同比下降 1.2%。
- **图中重点：**
  - ITO 仍是基本盘，但短期承压。
  - BPO 增长最明显，流程自动化和运营服务值得重点关注。
  - KPO 体量仍高，适合与 AI 数据分析、知识增强结合。
- **后续制图任务：**
  - 可做双轴：执行额 + 同比增速。
  - 需要注明数据口径为“离岸服务外包执行额”。

### 3.3 市场核心组成

#### 图 4：ITO / BPO / KPO 与 AI 服务融合图

- **建议标题：** AI Agent 与 AI 咨询如何横跨 ITO、BPO、KPO
- **推荐位置：** `1.3 市场核心组成` 后。
- **图表类型：** 三圆交集图 / 分层结构图。
- **图中内容：**
  - ITO：系统开发、接口集成、部署运维。
  - BPO：客服流程、工单流转、审批预处理、运营自动化。
  - KPO：知识整理、数据分析、辅助决策、行业研究。
  - AI Agent：位于三者交叉区，偏任务执行。
  - AI 咨询：覆盖诊断、PoC、治理、实施辅导和托管运营。
- **后续制图任务：**
  - 这是分析框架图，不需要新增数据。
  - 适合做成报告中最核心的解释图之一。

### 3.4 接单渠道与盈利占比分析

#### 图 5：渠道角色漏斗图

- **建议标题：** 技术外包获客渠道的角色分工
- **推荐位置：** `2.4 渠道选择建议` 后。
- **图表类型：** 漏斗图 / 渠道矩阵图。
- **图中内容：**
  - 国际平台：验证海外 AI 需求，适合增量线索。
  - 国内平台：适合试单和价格锚定，但竞争强。
  - 内容营销：建立信任，适合中长期获客。
  - 私域转介绍：转化率高，适合高信任项目。
  - 线下活动 / 园区 / 招投标：适合本地企业和长期合作。
- **可引用数据：**
  - Upwork AI-related work GSV 年化超过 3 亿美元，同比超过 50%。
  - Upwork AI Integration & Automation work 同比增长超过 90%。
  - Fiverr AI Agent 相关 freelancer searches 六个月增长 18347%，仅作为需求热度信号。
- **后续制图任务：**
  - 不建议画成“收入占比”，因为公开数据不足。
  - 更适合画“线索质量 x 转化周期 x 客单价潜力”的分析矩阵。

### 3.5 产品量级与盈利规模分析

#### 图 6：AI 咨询服务交付链路图

- **建议标题：** AI 咨询服务从诊断到托管运营的交付链路
- **推荐位置：** `3.3 AI 咨询服务` 后。
- **图表类型：** 流程图。
- **图中内容：**
  - AI 诊断
  - 场景筛选
  - ROI 评估
  - PoC / MVP
  - Agent / RAG 实施
  - 上线培训与治理
  - 月度托管优化
- **图中重点：**
  - 前端卖咨询，后端转实施和维护。
  - 服务收入从一次性诊断费延伸为项目费 + 月费。
- **后续制图任务：**
  - 不需要新增数据。
  - 可做成两层：交付动作 + 收费方式。

#### 图 7：产品量级与盈利对比图

- **建议标题：** 不同业务类型的客单价、周期与毛利率对比
- **推荐位置：** `产品量级与盈利对比表` 后。
- **图表类型：** 气泡图 / 雷达图 / 分组柱状图。
- **核心数据：**
  - 软件开发外包：8-30 万，1-3 个月，毛利率 35%-50%。
  - 数据分析外包：3-15 万，0.5-2 个月，毛利率 40%-50%。
  - AI Agent 外包：5-50 万 + 订阅费，1-2 个月，毛利率 50%-60%。
  - AI 咨询服务：2-20 万 + 月费，1 周-2 个月，毛利率 40%-60%。
- **图中重点：**
  - AI Agent 和 AI 咨询更适合形成高毛利与持续收入。
  - 软件开发仍可做，但要偏垂直场景定制。
- **后续制图任务：**
  - 先用区间中位数做图，图注保留区间。
  - 若正文缺少完整客单价区间，应先补充主报告正文和来源，再更新图表。

### 3.6 AI Coding 影响分析

- **推荐位置：** `4.5 对 3 人团队的经营启示` 后。
- **图表类型：** 三列表格图 / 影响矩阵。
- **图中内容：**
  - ITO：标准化开发被压价，系统集成和行业定制仍有价值。
  - BPO：流程执行向人机协同运营迁移。
  - KPO：低端资料整理被替代，高验证密度知识服务门槛提高。
- **后续制图任务：**
  - 这是分析图，不需要新增数据。
  - 可与“能力要求”结论复用。
